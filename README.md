# Tetris Dino Edition 🦕

A classic Tetris game built with C# and Windows Forms, featuring a dino-themed background.

## Gameplay

- 12×15 grid with all 7 standard tetrominoes (I, O, T, S, Z, L, J)
- Score tracker — earn 100 points per cleared line
- Elapsed time display
- Game over prompt with option to restart

## Controls

| Key | Action |
|-----|--------|
| ← → | Move left / right |
| ↓ | Soft drop |
| ↑ | Rotate |
| Space | Hard drop |

## Requirements

- Windows
- .NET Framework 4.7.2
- A `background.png` file in the working directory

## Running

Open `TetrisWinForms.sln` in Visual Studio and hit F5, or build and run the executable from `bin/Debug/`.
